# DriveDrop
## a cloud file sharing application with P2P encryption
### modules used:
- wxPython
- pyCryptoDome

### networking: socket programming

### Cryptography: 
- AES256,
- the key generated via the Diffie Helmen algorithm with RSA authentication to ensure no man in the middle attacks
